# 📊 Marketing Funnel & Conversion Analysis

## 🔍 Project Overview
This project analyzes marketing funnel performance to understand user conversion from contacts to customers and identify drop-off points.

## 📁 Dataset
Bank Marketing Dataset

## 🛠 Tools Used
- Power BI
- Excel

## 📊 Dashboard Features
- KPI Cards (Total Contacts, Customers, Conversion Rate)
- Funnel Chart (Contacts → Customers)
- Conversion by Contact Type
- Conversion by Job
- Conversion by Month

## 📈 Key Insights
- Significant drop-off between contacts and conversions
- Certain contact methods perform better
- Conversion varies across months
- Customer segments impact conversion rates

## 💡 Recommendations
- Focus on high-performing channels
- Improve targeting strategies
- Optimize campaign timing
